Is there any build step required?

Nope. This is a single HTML file using Tailwind via CDN. Just upload it.

Can I customize the colors and fonts?

Yes—update the Tailwind config object in the head or tweak classes inline.

How do forms work without a backend?

This demo validates client-side and shows a success toast. Connect to a service like Formspree for real submissions.
