How to create an Event QR code

  1. Enter Event Title (required) plus Description, Date, Time, Location, URL, and Contact.
  2. Customize colors and optionally upload a logo (keep it modest for best scanning).
  3. Click “Generate” to preview, then download as PNG or PDF.
  4. Test by scanning with multiple devices/apps and adjust contrast if needed.

Tip: Use a dark foreground on a light background. Increase size for distance viewing (posters/signage).

Best practices

  • High contrast is key (e.g., black on white). Avoid textured or low contrast backgrounds.
  • If using a logo, keep it within 10–25% of QR width and leave quiet zone margins.
  • Test on iOS and Android. Print a sample before mass printing.
  • Place QRs at eye level with clear call to action text near the code (e.g., “Scan for schedule”).
  • For “Add to Calendar,” link your QR to a hosted event page or ICS file for one tap imports.
