How to create a Wi Fi QR code

  1. Enter your Network Name (SSID).
  2. Select Security Type (WPA/WPA2/WPA3, WEP, or No Password).
  3. Enter the password if required; check “Hidden Network” if applicable.
  4. Customize colors and optionally upload a logo.
  5. Click “Generate,” then download PNG or PDF and test with iPhone/Android.

Tip: Use strong contrast (dark foreground on light background) for the best scan results.

Best practices

  • Use high contrast (e.g., black on white) and avoid busy backgrounds.
  • Keep logos modest (10–25% of QR width) to preserve scanability.
  • Increase size for print (at least 3–4 cm on the shortest side) and use 300 DPI.
  • Test on iOS and Android. Keep a plain (no logo) version as backup.
  • For hidden SSIDs, ensure the SSID is typed exactly; case and spacing matter.
