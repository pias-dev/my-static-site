How to use

  1. Type your expression or tap buttons. Use parentheses for clarity.
  2. Toggle DEG/RAD for trigonometric functions.
  3. Use % for quick percentage math (e.g., 200+10% → 220).
  4. Memory: M+ (add), M− (subtract), MR (recall), MC (clear).
  5. Press “=” or Enter to calculate. Use ←/→ to move the cursor inside the expression.

Examples

  • DEG mode: sin(30) → 0.5
  • Logs: log(1000) → 3, ln(e) → 1
  • Powers: 2^10 → 1024; x² and x³ buttons for quick squares/cubes
  • Roots: √(144) → 12; 27^(1/3) → 3 (ⁿ√y)
  • Factorial: 5! → 120
  • Percent: 200+10% → 220, 200-10% → 180
